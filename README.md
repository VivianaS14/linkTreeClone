# LinkTree Clone

## Creations commands

- Create Controller-View

```
rails g controller <name> <functionName>
```

- Create SCAFFOLD -> create model, view & controller for a CRUD (Create, Read, Update, Delete)

```
rails g scaffold <name> <attribute:type> <attribute:type>...
```

- Then run

```
rails db:migrate
```
